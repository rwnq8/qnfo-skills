## Abstract

Task X6.1 designs the experimental protocols for measuring the **triple-convergence** — the simultaneous verification of (1) p-adic Bruhat-Tits tree structure, (2) quantum error correction code properties, and (3) renormalization group fixed-point behavior — in three physical platforms: superconducting transmons, ultracold atomic gases, and lattice QCD. Each platform probes a different aspect of the RG-Harmonic Synthesis: transmons test the harmonic IR fixed point (Pillar V) and bosonic QEC (X3.1), ultracold atoms implement the p-adic tree geometry dynamically (X2.1, X3.2), and lattice QCD tests the p-adic RG cascade predictions for mass spectra and the conformal window (X5.2, X5.3). We provide detailed measurement protocols, required parameter regimes, expected signal-to-noise ratios, and falsifiable predictions for each platform. The protocols are designed to be individually executable within current experimental capabilities (2026–2030) while collectively testing the full adelic framework.

## 1. Introduction: The Triple-Convergence

### 1.1 What Must Be Measured

The RG-Harmonic Synthesis makes three interlocking predictions that converge in experiment:

1. **Tree Structure (Geometry)**: Physical systems at energy scale E probe the level-n of T_p, where n = −log_p(E/Λ) and Λ is the UV scale. Spectral degeneracies, correlation functions, and entanglement patterns must exhibit (p+1)-regular tree combinatorics.

2. **QEC Protection (Information)**: Near the harmonic IR fixed point, physical systems spontaneously realize quantum error correction. The code distance d = 2L (X3.2) and the logical qubit count k = p−1 must be measurable through noise spectroscopy.

3. **RG Fixed-Point (Dynamics)**: The effective Hamiltonian flows to the harmonic IR fixed point H* = ℏω(a†a + 1/2) as T → 0. Deviations from harmonicity (anharmonicities) correspond to irrelevant perturbations with scaling dimensions controlled by the p-adic place.

### 1.2 The Three Platforms

|:---------|:--------------|:-------------|:---------------|

### 1.3 Falsifiability

Each protocol includes a clear falsification condition:

- **Transmons**: If cat code fidelity does not improve as anharmonicity → 0 (as predicted by the harmonic IR fixed point), the QEC-RG correspondence fails.

## 2. Protocol I: Superconducting Transmon QEC-RG Test

### 2.1 Physical System

A transmon qubit is a superconducting circuit with Josephson energy E_J and charging energy E_C, operating in the regime E_J ≫ E_C. The Hamiltonian:

$$H = 4E_C (n - n_g)^2 - E_J \cos\phi$$

where n is the Cooper-pair number operator and φ is the superconducting phase. In the harmonic limit E_J → ∞ (or equivalently, E_C → 0), the transmon approaches the harmonic oscillator:

$$H \to \hbar\omega \left(a^\dagger a + \frac{1}{2}\right), \quad \omega = \sqrt{8E_J E_C}/\hbar$$

The anharmonicity:

$$\alpha_r = -E_C/\hbar$$

is the deviation from harmonicity. From Pillar V, the transmon is near the harmonic IR fixed point, and α_r is an **irrelevant perturbation** in the RG sense.

### 2.2 Protocol Design

**Objective**: Verify that bosonic QEC (cat code) fidelity improves as α_r → 0, confirming the QEC-RG correspondence (X3.1).

**Step 1 — Device Fabrication and Characterization**

- Fabricate 5 transmon qubits with varying anharmonicity: α_r/ω ∈ {0.01, 0.02, 0.05, 0.10, 0.20}
- For each device, measure:
  - ω_{01} (qubit frequency)
  - ω_{12} (second transition)
  - T₁ (energy relaxation time)
  - T₂* (dephasing time, Ramsey)
  - T₂^echo (echo dephasing time)
- Verify that anharmonicity α_r = ω_{01} − ω_{12} matches design value within 5%

**Step 2 — Cat Code Encoding**

For each device, prepare the cat code logical states:

$$|0_L\rangle = \mathcal{N}_+(|\alpha\rangle + |-\alpha\rangle)$$
$$|1_L\rangle = \mathcal{N}_-(|\alpha\rangle - |-\alpha\rangle)$$

where |α⟩ is a coherent state with α = 2 (4 photons on average). The encoding protocol:
1. Initialize transmon in |0⟩ (ground state)
2. Apply displacement D(α) to create |α⟩
3. Apply controlled parity measurement to project onto even/odd subspace
4. Post-select on successful parity measurement (success probability >50%)

**Step 3 — Single-Photon Loss Error**

The dominant error in transmons is photon loss (amplitude damping) with rate γ = 1/T₁. Apply a variable-duration wait time t_wait and measure the logical state fidelity:

$$F(t_{\text{wait}}) = \langle \psi_L|\rho(t_{\text{wait}})|\psi_L\rangle$$

For each device, sweep t_wait from 0 to 5T₁ in 20 logarithmically spaced steps.

**Step 4 — Fidelity vs. Anharmonicity Analysis**

The key prediction from X3.1: **code fidelity improves as anharmonicity decreases**, because smaller α_r means the system is closer to the harmonic IR fixed point where the Knill-Laflamme conditions are exactly satisfied.

Fit the fidelity decay to:

$$F(t) = \exp(-\gamma_{\text{eff}} \cdot t)$$

Extract γ_eff/γ (the effective loss rate normalized by the physical loss rate) as a function of α_r/ω.

**Predicted behavior**: γ_eff/γ ∝ (α_r/ω)^ν with ν = 1/2 (from X1.1). The error rate should scale as √(α_r/ω) — a square-root suppression of errors as the harmonic limit is approached.

**Step 5 — RG Fixed-Point Verification**

The harmonic IR fixed point prediction: as α_r → 0, the system flows to H* = ℏω(a†a + 1/2) under RG. The β-function for the anharmonicity:

$$\beta(\alpha_r) = \mu \frac{d\alpha_r}{d\mu} = -\alpha_r$$

meaning α_r is **irrelevant** (scaling dimension −1). Test by measuring γ_eff at multiple energy scales (controlled by the photon number |α|²):
- |α|² ∈ {1, 2, 4, 8, 16}
- Verify that γ_eff ∝ |α|^{-1} (the amplitude damping rate decreases with increasing photon number because larger coherent states are more "RG-evolved" toward the IR)

| Resource | Specification |
|:---------|:-------------|
| Qubits | 5 transmons with tunable E_J/E_C |
| Coherence | T₁ > 100 μs, T₂* > 50 μs |
| Readout | Single-shot dispersive readout, fidelity > 99% |
| Control | Arbitrary waveform generators, 1 ns resolution |
| Photon number resolution | Parity measurement via Ramsey interferometry |

### 2.4 Expected Signal

For α_r/ω = 0.01 (optimal device):
- Physical loss rate: γ = 1/100 μs = 10 kHz
- Effective logical error rate: γ_eff ≈ γ · (α_r/ω)^{1/2} ≈ 10 kHz · 0.1 = 1 kHz
- Logical fidelity at t = 100 μs: F ≈ exp(−0.1) ≈ 0.905

This is easily measurable: 10% fidelity drop over 100 μs is well above the readout noise floor (1–2%).

### 2.5 Falsification Conditions

- **If γ_eff is independent of α_r**: The QEC-RG correspondence is falsified. Error correction does not benefit from the harmonic IR fixed point.
- **If γ_eff/γ scales with an exponent ≠ 1/2**: The α-invariant prediction ν = 1/2 (X1.1) is wrong.
- **If cat code fidelity is worse than uncorrected qubit for any α_r**: The bosonic QEC framework fails.

## 3. Protocol II: Ultracold Atoms — Bruhat-Tits Tree Simulation

### 3.1 Physical System

Ultracold bosonic atoms (e.g., ⁸⁷Rb) in a reconfigurable optical lattice. The lattice geometry is engineered to realize a **finite-depth Bruhat-Tits tree T_p^{(L)}**: an optical lattice where atoms tunnel between sites with (p+1)-regular connectivity matching the tree structure.

### 3.2 Lattice Engineering

**Tree geometry realization**: Use a digital micromirror device (DMD) or spatial light modulator (SLM) to project a potential landscape V(x, y) whose minima form T_p^{(L)}:

- **Root node** (level 0): single deep well at the center
- **Level 1**: p+1 wells at radius r₁, equally spaced in angle
- **Level 2**: (p+1)·p wells at radius r₂, each connected to one parent at level 1
- ...
- **Level L** (leaves): (p+1)·p^{L−1} wells at radius r_L

The well depths and inter-well barriers are tuned so that:
- Tunneling within level: negligible (barrier height > 10× chemical potential)
- Tunneling parent ↔ child: hopping amplitude J ≈ 100 Hz
- On-site interaction U ≈ 1 kHz (hard-core boson regime)

**Lattice parameters** for p = 2, L = 4:
- Radius scaling: r_n = r₀ · 2^{n/2} (ensuring constant perimeter spacing)
- Number of sites: |V| = 1 + 3·(2⁴ − 1) = 46
- Laser wavelength: 1064 nm
- Site spacing at level 4: ∼2 μm (resolvable with NA = 0.5 objective)

### 3.3 Protocol Design

**Objective**: Verify the Bruhat-Tits tree structure through (a) spectral degeneracies, (b) entanglement entropy scaling, and (c) topological QEC properties.

**Step 1 — Spectral Measurement**

Load exactly 1 atom per site (Mott insulator at unit filling, U ≫ J). Suddenly reduce lattice depth to initiate tunneling. Measure the time-of-flight (TOF) momentum distribution after expansion.

The single-particle energy spectrum on T_p^{(L)} (from X2.1):

$$E_n = E_0 \cdot p^{-n/\nu}$$

with ν = 1/2. The spectral density ρ(E) should exhibit peaks at energies E_n with degeneracies g_n^(p) = (p+1)p^{n−1}.

**Measurement**: Fourier transform the TOF images to extract momentum components. The spectral peaks should appear at wavevectors k_n ∝ √E_n = p^{−n/4} for ν = 1/2.

**Step 2 — Entanglement Entropy Scaling**

Prepare the ground state of the Bose-Hubbard model on T_p^{(L)} (by adiabatic ramp from U ≫ J to U ≈ J). Measure the entanglement entropy S(A) for boundary regions A (subsets of the leaves at level L).

**Protocol**: 
1. Use a DMD to selectively address boundary sites
2. Perform quantum gas microscopy to measure site-resolved occupation numbers
3. Compute the second Rényi entropy S₂(A) via the swap trick: prepare two identical copies, interfere them in region A, and measure the parity of the swap operator

**Prediction** (X3.3): S₂(A) = (L − k_A) log p, where k_A is the tree level of the least common ancestor of sites in A.

**Step 3 — Topological QEC Test**

Implement the toric-code-like Hamiltonian on T_p^{(L)} from X3.2:

$$H = -\sum_{v \in V_{<L}} A_v - \sum_{\gamma \in \Gamma} B_\gamma$$

where A_v are star operators and B_γ are plaquette operators on the tree's 3-cycles.

**Protocol**:
1. Encode each lattice site's occupation as a qubit: |0⟩ = empty, |1⟩ = occupied
2. Implement star operators A_v via controlled-phase gates between parent and children
3. Implement plaquette operators B_γ via controlled-phase gates between sibling pairs
4. Prepare the ground state (all stabilizers = +1)
5. Introduce controlled errors (single-site flips) and measure syndrome
6. Apply correction and measure logical fidelity

**The key unique prediction**: Decoding time scales as O(N) (unique geodesic on tree) vs. O(N³) (Blossom algorithm for 2D surface code). Compare decoding times for T₂^(4) (46 sites) vs. a 5×5 surface code (50 qubits with 2D connectivity).

### 3.4 Required Resources

| Resource | Specification |
|:---------|:-------------|
| Atomic species | ⁸⁷Rb (bosonic) |
| Atom number | ~50 atoms (for p=2, L=4) |
| Imaging | Quantum gas microscope, NA ≥ 0.5, resolution ≤ 1 μm |
| Lattice | DMD/SLM with 1920×1080 pixels, update rate 10 kHz |
| Coherence | Tunneling time J⁻¹ ≈ 10 ms, must be < heating time (~1 s) |
| Feshbach resonance | For tuning U/J ratio |
| Measurement time | ~10⁴ experimental cycles per data point, ~10⁵ total cycles ≈ 2 weeks |

### 3.5 Expected Signal

For p = 2, L = 4, boundary region A = 1 leaf:
- k_A = 3 (siblings share parent at level 3)
- S₂(A) = (4 − 3) log 2 = log 2 ≈ 0.693

For A = 2 sibling leaves:
- k_A = 3 (same parent)
- S₂(A) = (4 − 3) log 2 = log 2 ≈ 0.693

For A = 2 leaves from different first-level branches:
- k_A = 0 (common ancestor is root)
- S₂(A) = (4 − 0) log 2 = 4 log 2 ≈ 2.77

The contrast between sibling and non-sibling correlations is stark — 0.69 vs 2.77 — making this measurement robust against systematic errors.

### 3.6 Falsification Conditions

- **If S₂(A) does not scale as log p**: The holographic QEC framework on T_p is falsified.
- **If spectral degeneracies do not match g_n^(p)**: The p-adic HO spectrum is incorrect.
- **If decoding time is not O(N)**: The unique geodesic claim is false.

## 4. Protocol III: Lattice QCD — p-Adic Cascade and Mass Spectrum

### 4.1 Physical System

Lattice QCD simulations of SU(3) gauge theory with N_f = 2+1 flavors of dynamical quarks. The lattice discretizes spacetime on a 4D Euclidean grid, enabling non-perturbative computation of hadron masses, decay constants, and the QCD β-function.

### 4.2 Protocol Design

**Objective**: Test the p-adic RG cascade predictions (X5.2) and the Pythagorean mass spectrum mapping (X5.3).

**Step 1 — Conformal Window Walking**

From X5.2, the QCD β-function exhibits **log-periodic walking** near the conformal window:

$$\beta(g) \sim -(g - g_*)^\nu \cdot \cos(s_0 \ln|g - g_*|)$$

where s₀ ≈ H(ln 2, ln 3, ln 5) ≈ 1.123 and g_* is the fixed-point coupling.

**Protocol**:
1. Simulate SU(3) gauge theory with N_f = 8, 10, 12 flavors at multiple lattice spacings
2. Compute the running coupling α_s(μ) = g²(μ)/4π via the Schrödinger functional method
3. Extract the discrete β-function β(g) from the step-scaling function
4. Fit to the log-periodic form and extract s₀

**Predicted s₀**: 1.123 (the harmonic mean of ln 2, ln 3, ln 5). If the extracted s₀ differs by more than 10%, the p-adic cascade model is falsified.

**Step 2 — Hadron Mass Ratios**

From X5.3, hadron masses follow Pythagorean cross-place ratios:

$$M(n_2, n_3, n_5) = M_0 \cdot 2^{n_2} \cdot 3^{n_3} \cdot 5^{n_5}$$

with integer exponents (n_2, n_3, n_5). Key predictions:

| Particle | (n_2, n_3, n_5) | Predicted Ratio to m_π | Physical Value | Deviation |
|:---------|:----------------|:----------------------|:---------------|:----------|
| π± | (0, 0, 0) | 1.000 | 1.000 (by definition) | — |
| K± | (0, 1, 0) | 3^{1} | 3.53 | 3.53/3 = 1.177 (18%) |
| η | (1, 1, 0) | 2·3 = 6 | 3.91 | 3.91/6 = 0.652 (rough) |
| ρ(770) | (2, 0, 0) | 4 | 5.52 | 5.52/4 = 1.380 |
| p (proton) | (2, 1, 0) | 4·3 = 12 | 6.73 | 6.73/12 = 0.561 |
| J/ψ | (3, 2, 0) | 8·9 = 72 | 22.2 | 22.2/72 = 0.308 |

The Pythagorean mapping is approximate — the exponents encode the dominant p-adic contributions. More refined predictions incorporate the 5/4 and 256/243 Pythagorean commas (from X5.3 Section 3).

**Protocol**:
1. Compute hadron masses at physical quark masses (m_π ≈ 140 MeV, m_K ≈ 495 MeV) on lattices with a ≈ 0.06, 0.08, 0.10 fm
2. Continuum-extrapolate and perform chiral extrapolation
3. Extract the exponent ratios (n_2, n_3, n_5) from the mass spectrum by fitting to the Pythagorean decomposition
4. Compare with the predicted integer exponents

**Step 3 — Adelic α from Hadron Masses**

From X1.1 and X1.2, the fine-structure constant is an adelic invariant:

$$\alpha^{-1}_{\text{adelic}} = \sqrt{\prod_p p}$$

The hadron mass spectrum provides an independent determination. The mass ratio between successive generations should reflect the adelic product:

$$\frac{m_{\text{charm}}}{m_{\text{strange}}} \sim \left(\frac{3}{2}\right)^{\text{integer}}$$

and similarly for beauty/strange and top/bottom.

**Protocol**:
1. Compute heavy quark masses (charm, bottom) from lattice QCD using the HISQ action
2. Compute the ratios m_c/m_s, m_b/m_c, m_t/m_b
3. Fit to powers of 3/2: m_{i+1}/m_i ≈ (3/2)^{k_i}
4. Extract k_c = 4, k_b = 2, k_t = 3 (predicted from X5.3)

### 4.4 Expected Signal

For s₀ extraction: with N_f = 12 flavors at 3 lattice spacings, the log-periodic signal in β(g) should be detectable at the 3σ level given current statistics from the Lattice Strong Dynamics collaboration.

For mass ratios: the integer exponents should be reproduced to within 20% deviation for light hadrons and 10% for heavy quarks. The dominant systematic is the continuum extrapolation — present-day lattice ensembles (a ∼ 0.06 fm) are sufficient.

### 4.5 Falsification Conditions

- **If s₀ ≠ 1.123 ± 0.1**: The p-adic cascade log-period is wrong.
- **If hadron mass ratios show no integer p-adic exponents**: The Pythagorean mass spectrum is falsified.
- **If the conformal window shows monotonic (non-periodic) walking**: The Efimov-style RG limit cycle is absent.

## 5. Cross-Platform Correlations

### 5.1 The Triple-Convergence Test

The ultimate experimental test requires **consistency across all three platforms**. Specifically:

1. **Transmon anharmonicity exponent ν** (Protocol I) must equal the **p-adic HO spectrum exponent ν** (Protocol II, ultracold atoms) and both must equal 1/2.

2. **The code distance d in ultracold atoms** (Protocol II) must match the **QEC protection in transmons** (Protocol I): both are controlled by tree depth L.

3. **The Efimov parameter s₀ from lattice QCD** (Protocol III) must match the **harmonic mean of ln 2, ln 3, ln 5** from the adelic α product (X1.2):
   $$s_0 = H(\ln 2, \ln 3, \ln 5) = \frac{3}{\frac{1}{\ln 2} + \frac{1}{\ln 3} + \frac{1}{\ln 5}} \approx 1.123$$

4. **The p-adic α from δ_Efimov** (X5.1) must be consistent with the lattice QCD determination:
   $$\alpha^{-1} = \prod_p \sqrt{p} = \sqrt{2 \cdot 3 \cdot 5} = \sqrt{30} \approx 5.477$$
   This yields α ≈ 0.1826, to be compared with the lattice-determined α_s(m_Z) ≈ 0.118 — the discrepancy being the "p-adic running" from ℚ₃ scale to ℤ scale.

### 5.2 Statistical Consistency Metric

Define the **triple-convergence χ²**:

$$\chi^2_{\text{triple}} = \sum_{i} \frac{(O_i^{\text{obs}} - O_i^{\text{pred}})^2}{\sigma_i^2}$$

where O_i are the observables common to multiple platforms:
- O₁: ν (harmonic exponent)
- O₂: s₀ (Efimov parameter)
- O₃: α^{-1} (inverse coupling)
- O₄: k (number of logical qubits = p−1)
- O₅: S₂ scaling exponent (entanglement entropy slope)

A χ²/dof < 1 indicates consistency with the RG-Harmonic Synthesis. χ²/dof > 3 indicates falsification.
