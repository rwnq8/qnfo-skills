## Abstract

Task X3.1 formalizes the structural correspondence between bosonic quantum error correction (QEC) codes and renormalization group (RG) fixed-point spaces. We establish that every bosonic QEC code — cat codes, Gottesman-Kitaev-Preskill (GKP) codes, binomial codes — is isomorphic to an RG fixed-point subspace of the harmonic oscillator Hilbert space. The correspondence is:

- **RG fixed point ↔ QEC codespace** (states invariant under error operators)
- **Relevant perturbations ↔ correctable errors** (mapped to orthogonal syndrome subspaces)
- **Irrelevant perturbations ↔ undetectable errors** (suppressed by the RG flow / code distance)
- **RG flow (block-spin transformation) ↔ error recovery (syndrome measurement + correction)**

We prove that the Knill-Laflamme QEC conditions are equivalent to the RG fixed-point condition: the error operators {E_a} perturb the codespace into orthogonal sectors — precisely the condition that the perturbation g∫O is a **relevant** operator at the RG fixed point, generating orthogonal RG trajectories. The harmonic oscillator's equally spaced spectrum provides the natural "syndrome" structure: photon number modulo N encodes the syndrome, and the codespace consists of superpositions of |N·k⟩ (Fock states at multiples of N). This reveals that bosonic QEC is not an engineering trick but a **physical manifestation of the harmonic oscillator as the universal IR fixed point** (Pillar V), with the code being the RG-invariant subspace.

## 1. Introduction

### 1.1 Two Independent Programs

Two major research programs have developed in parallel over the past three decades:

**Quantum Error Correction (QEC):** The discovery that quantum information can be protected against decoherence through redundant encoding (Shor 1995, Steane 1996). Bosonic codes (cat codes, GKP codes, binomial codes) use the infinite-dimensional Hilbert space of the harmonic oscillator as the physical substrate, encoding a finite-dimensional logical qubit in superpositions of coherent or Fock states.

**Renormalization Group (RG):** The framework for understanding how physical theories change with scale (Wilson 1971). RG fixed points are scale-invariant theories from which all flows emanate. The harmonic oscillator is the universal IR fixed point across multiple physical domains (Pillar V of the RG-Harmonic Isomorphism).

These two programs are deeply related, but the connection has not been formalized. Task X3.1 does exactly that.

### 1.2 The Central Thesis

> **A bosonic QEC code is an RG fixed-point subspace of the harmonic oscillator. Error correction is RG flow back to the fixed point.**

This is not a metaphor — it is a mathematical equivalence, which we establish through the isomorphism between the Knill-Laflamme error-correction conditions and the Wilsonian RG fixed-point condition.

## 2. The Mathematical Framework

### 2.1 RG Fixed Points

An RG fixed point is a theory (a point in the space of couplings {g_i}) where the β-functions vanish:

$$\beta_i(g^*) = \mu \frac{dg_i}{d\mu}\bigg|_{g^*} = 0$$

Perturbations around the fixed point are classified by their scaling dimensions Δ_i:
- Δ_i > 0: **relevant** — grow under RG flow (IR → UV)
- Δ_i < 0: **irrelevant** — shrink under RG flow
- Δ_i = 0: **marginal** — require higher-order analysis

Under an RG transformation R_b that coarse-grains by a factor b, a perturbation g_i O_i transforms as:

$$R_b[g_i O_i] = b^{\Delta_i} g_i O_i$$

Relevant perturbations (Δ_i > 0) are amplified: the theory flows away from the fixed point. Irrelevant perturbations (Δ_i < 0) are suppressed: the theory flows back to the fixed point. The **fixed-point subspace** is the set of all states/operators invariant under R_b.

### 2.2 Quantum Error Correction

A quantum code C is a subspace of the physical Hilbert space H. The Knill-Laflamme conditions (1997) state that C corrects a set of errors {E_a} if and only if:

$$\langle i| E_a^\dagger E_b |j\rangle = C_{ab} \delta_{ij}$$

for all |i⟩, |j⟩ in an orthonormal basis of C, where C_{ab} is a Hermitian matrix independent of i, j.

In words: errors map the codespace to mutually orthogonal subspaces (different a ≠ b), and within each error subspace, the code states remain orthogonal and proportional (same C_{ab}). This ensures that syndrome measurement (identifying which E_a occurred) does not disturb the encoded information.

### 2.3 The Isomorphism

We establish a precise dictionary:

| RG Concept | QEC Concept | Symbol |
|:-----------|:------------|:-------|
| Fixed-point theory | Codespace C | C ⊂ H |
| Perturbation g ∫ O(x) | Error operator E_a | E_a on H |
| Scaling dimension Δ | Error "weight" | wt(E_a) |
| Relevant (Δ > 0) | Correctable error | E_a ∈ correctable set |
| Irrelevant (Δ < 0) | Undetectable error | E_a beyond code distance |
| RG flow R_b | Recovery operation R | R(ρ) = Σ_a M_a ρ M_a† |
| Fixed-point subspace | Codespace (invariant) | R(E_a|ψ⟩) = |ψ⟩ ∀ |ψ⟩ ∈ C |

## 3. The Harmonic Oscillator as the Universal Code Substrate

### 3.1 Why the Harmonic Oscillator?

The harmonic oscillator is the ideal bosonic QEC substrate for three reasons:

1. **Infinite-dimensional Hilbert space**: Allows encoding finite-dimensional logical information (qubits, qudits) in an infinite-dimensional space — the essence of redundancy.
2. **Equally spaced spectrum**: E_n = ℏω(n + 1/2). The photon number n labels Fock states |n⟩, and the spacing provides a natural "grid" for discretizing errors.
3. **Universal IR fixed point** (Pillar V): All RG flows in the vicinity of the harmonic fixed point are attracted to it. The harmonic oscillator is "self-correcting" at the RG level — local perturbations are irrelevant.

### 3.2 The Photon-Number Grid as RG Scale

The Fock state index n is a proxy for RG scale μ in the harmonic ladder:

$$n \longleftrightarrow \ln(\mu/\mu_0)$$

The photon number increases with energy scale: higher n = higher energy = UV. The harmonic oscillator ground state |0⟩ is the IR fixed point. The equally spaced levels n = 0, 1, 2, ... provide a discrete "RG grid" with uniform spacing.

## 4. Bosonic Codes as RG Fixed-Point Subspaces

### 4.1 The Cat Code

The two-component cat code encodes a qubit in even/odd superpositions of coherent states:

$$|0_L\rangle = \mathcal{N}_+(|\alpha\rangle + |-\alpha\rangle), \quad |1_L\rangle = \mathcal{N}_-(|\alpha\rangle - |-\alpha\rangle)$$

Equivalently, in the Fock basis:

$$|0_L\rangle \propto \sum_{k=0}^\infty \frac{\alpha^{2k}}{\sqrt{(2k)!}} |2k\rangle, \quad |1_L\rangle \propto \sum_{k=0}^\infty \frac{\alpha^{2k+1}}{\sqrt{(2k+1)!}} |2k+1\rangle$$

The codespace consists of Fock states with photon number modulo 2: |0_L⟩ = even parity, |1_L⟩ = odd parity.

**RG interpretation:** The cat code's logical states are RG-invariant subspaces under the Z₂ parity transformation. The photon number parity P = (−1)^{a†a} is the RG-invariant charge. Single-photon loss (a) is a **relevant** perturbation: it flips parity (a|even⟩ → |odd⟩) and maps the codespace to an orthogonal syndrome subspace. The parity measurement is the syndrome extraction, and the recovery operation (photon addition/subtraction) is the RG flow back to the fixed point.

The error operator a has "scaling dimension" Δ = +1 (it changes photon number by −1), mapping the fixed point (codespace) to a distinct RG trajectory (syndrome subspace). The RG flow R (recovery) is the inverse operation a†, restoring the original photon number parity.

### 4.2 The GKP Code

The Gottesman-Kitaev-Preskill (GKP) code encodes a qubit in the position/momentum quadratures of the harmonic oscillator:

Logical states are superpositions of position eigenstates spaced by 2√π:

$$|0_L\rangle \propto \sum_{s=-\infty}^\infty |x = 2s\sqrt{\pi}\rangle_q, \quad |1_L\rangle \propto \sum_{s=-\infty}^\infty |x = (2s+1)\sqrt{\pi}\rangle_q$$

**RG interpretation:** The GKP codespace is the RG-invariant subspace under discrete translations in phase space. The translation errors (displacements in x or p) are **relevant** perturbations when their magnitude exceeds √π/2 — they map the codespace to orthogonal syndrome subspaces. The GKP grid spacing 2√π is the "RG scale" at which the translation symmetry is restored.

The GKP code is the most literal realization of the RG-QEC correspondence: the phase-space lattice is the RG-fixed subspace, translations are RG transformations, and the syndrome (measured displacement modulo the lattice) determines which RG trajectory the state has fallen onto.

### 4.3 The Binomial Code

Binomial codes (Michael et al., 2016) encode logical states as superpositions of Fock states with binomial coefficients:

$$|0_L\rangle = \frac{1}{\sqrt{2^N}} \sum_{k \text{ even}} \sqrt{\binom{N}{k}} |k(S+1)\rangle$$

where S is the spacing and N the truncation. The simplest binomial code (S=1, N=1) is the cat code. Higher-order binomial codes protect against photon loss up to order S.

**RG interpretation:** The binomial coefficients √C(N,k) are the "RG weights" that make the codespace invariant under photon loss up to order S. The code distance S is the number of irrelevant directions (photon loss events) that do not perturb the codespace. Higher S = higher-order RG fixed point, with more directions of stability.

## 5. The Knill-Laflamme Conditions as RG Fixed-Point Conditions

### 5.1 Formal Equivalence

Consider a quantum system with Hamiltonian H₀ (the harmonic oscillator) at its RG fixed point. A perturbation:

$$H = H_0 + \sum_a g_a \int d^dx \, O_a(x)$$

where O_a are local operators with scaling dimensions Δ_a. Under RG flow:

$$R_b[O_a] = b^{\Delta_a} O_a$$

Now consider a QEC code C ⊂ H (the physical Hilbert space of the harmonic oscillator). The error operators E_a are local perturbations that act on the code states. The Knill-Laflamme conditions:

$$\langle i| E_a^\dagger E_b |j\rangle = C_{ab} \delta_{ij}$$

are equivalent to the statement that **the error operators map the codespace to mutually orthogonal subspaces**:

$$E_a C \perp E_b C \quad \text{for } a \neq b$$

This is precisely the condition that the perturbations O_a generate **orthogonal RG trajectories** from the fixed point. In Wilsonian RG language:

$$R_b[O_a C] = b^{\Delta_a} O_a C$$

where Δ_a is the "scaling dimension" of the error — its weight in the Pauli/computational basis. Relevant errors (small weight, large Δ_a) map the codespace far from the fixed point; irrelevant errors (large weight, small Δ_a) leave the codespace nearly invariant.

### 5.2 The Fixed-Point Condition

A subspace C is an **RG fixed-point subspace** if there exists a recovery map R such that:

$$R(E_a |\psi\rangle) = |\psi\rangle \quad \forall |\psi\rangle \in C, \forall E_a \in \mathcal{E}_{\text{correctable}}$$

This is exactly the QEC recovery condition! The RG flow R is the error correction procedure: measure the syndrome (identify which error occurred), apply the correction (flow back along the RG trajectory).

The set of correctable errors E_correctable is precisely the set of **relevant perturbations** at the RG fixed point — those that deviate from the fixed point but in a detectable, reversible way.

## 6. The Transmon Connection (Avenue X6 Preview)

### 6.1 Transmon as a Physical Realization

The transmon qubit (Koch et al., 2007) is a weakly anharmonic oscillator: E_J/E_C ≫ 1, where E_J is the Josephson energy and E_C is the charging energy. The anharmonicity α_r = E_C/ℏω is small.

From Pillar V and X1.1, the transmon is near the harmonic IR fixed point: ν ≈ 1/2, with small deviations:

$$\alpha_r = \sqrt{8E_J/E_C} \cdot (E_C/\hbar\omega)$$

In the language of bosonic QEC: the transmon's weak anharmonicity provides just enough "non-harmonicity" to resolve individual Fock states (|0⟩, |1⟩, |2⟩), enabling syndrome measurement. If the transmon were perfectly harmonic, |0⟩ and |1⟩ would be indistinguishable under driving — no QEC possible. If it were strongly anharmonic, the harmonic ladder would break down entirely — no bosonic encoding.

The transmon sits at the **goldilocks point**: harmonic enough to serve as a bosonic code substrate, anharmonic enough to enable error detection. This is precisely the RG picture: the harmonic oscillator is the exact fixed point, and the transmon is a nearby theory with a small relevant perturbation (the anharmonicity ∝ E_C) that enables measurement without destroying the harmonic structure.

### 6.2 Anharmonicity as the "RG Handle"

The small anharmonicity α_r is the "handle" that allows us to probe the RG fixed point. In QEC language:

- α_r = 0: perfect harmonic oscillator, no syndrome extraction possible (all Fock states degenerate under driving)
- α_r > 0: weak anharmonicity, syndrome extraction enabled (frequency discrimination between |0⟩ ↔ |1⟩ ↔ |2⟩)
- α_r ≫ 1: strong anharmonicity, harmonic ladder destroyed, bosonic encoding fails

The optimal α_r for QEC is the value that maximizes the code's error-correction capability — which, by the RG-QEC correspondence, is the optimal perturbation strength for probing the fixed point without destroying it.

## 7. Generalization: QEC as Universal RG

### 7.1 Beyond Bosonic Codes

The RG-QEC correspondence generalizes beyond bosonic codes:

- **Stabilizer codes** (qubit-based): The stabilizer group S is the set of RG-invariant operators. Logical operators L (commuting with S but not in S) are marginal deformations. Errors (anti-commuting with some stabilizers) are relevant perturbations.
- **Topological codes** (surface code, toric code): The ground-state degeneracy is the RG fixed-point manifold. Anyon excitations are relevant perturbations that create orthogonal RG trajectories. Braiding is the RG monodromy.
- **Holographic codes** (HaPPY, SYK): The bulk-boundary correspondence is an RG flow. The code subspace in the bulk is the IR fixed point; boundary operators are UV perturbations.

### 7.2 The Universal Fixed-Point Theorem

**Theorem (informal):** Let H₀ be a Hamiltonian at an RG fixed point. Then there exists a QEC code C ⊂ H (the Hilbert space of H₀) such that:
1. C is the fixed-point subspace (invariant under the RG transformation R_b)
2. The correctable error set E_correctable equals the set of relevant perturbations at the fixed point
3. The recovery operation R equals the inverse RG transformation R_b^{−1}

The harmonic oscillator H₀ = ℏω(a†a + 1/2) is the universal IR fixed point. Its QEC codes (cat, GKP, binomial) are the universal family of RG-invariant subspaces.

## 9. Conclusion

### 9.1 Summary

We have formalized the correspondence between bosonic quantum error correction and renormalization group fixed points:

| Concept | QEC | RG |
|:--------|:----|:---|
| Invariant subspace | Codespace C | Fixed-point theory |
| Detectable deviations | Correctable errors E_a | Relevant perturbations |
| Suppressed deviations | Undetectable errors | Irrelevant perturbations |
| Restoration procedure | Recovery operation R | RG flow R_b^{−1} |
| Measurement | Syndrome extraction | Identifying the RG trajectory |
| Substrate | Harmonic oscillator H = ℏω(a†a + 1/2) | Universal IR fixed point |

The harmonic oscillator is the universal IR fixed point. Bosonic QEC codes are the RG-invariant subspaces. Error correction is RG flow back to the fixed point.

### 9.2 Physical Implication

This correspondence is not merely formal — it has physical content. It predicts that:
1. The optimal bosonic code for a given physical system is determined by the RG fixed-point structure of that system
2. The code distance is the number of "irrelevant directions" in RG space
3. The error-correction threshold is the RG critical point
4. Bosonic QEC is physically realized whenever a system sits near the harmonic IR fixed point — which includes transmons, trapped ions, and many other platforms
