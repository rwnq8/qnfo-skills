## 3. Computing ∏_p α_p

### 3.1 The Infinite Product

With α_p = p^{-1/2}, the product over all primes diverges in the naive sense:

$$\prod_{p} p^{-1/2} = \exp\left(-\frac{1}{2}\sum_p \ln p\right) \to 0$$

The sum ∑_p ln p diverges logarithmically (the Chebyshev function θ(x) ~ x). Regularization is required.

### 3.2 Zeta Regularization

The standard tool for regularizing divergent products over primes is zeta regularization. For an arithmetic function f(p), the zeta-regularized product is defined as:

$$\prod_p f(p) \equiv \exp\left(-\frac{d}{ds}\sum_p f(p)^s \ln f(p)\bigg|_{s=0}\right)$$

or equivalently, through the analytic continuation of the prime zeta function P(s) = Σ_p p^{-s}.

For f(p) = p, the known zeta-regularized product is:

$$\boxed{\prod_p p = 4\pi^2}$$

This is a standard result in analytic number theory, related to the functional equation of the Riemann zeta function. The zeta-regularized product of all primes is 4π².

Therefore:

$$\prod_p p^{-1/2} = \left(\prod_p p\right)^{-1/2} = (4\pi^2)^{-1/2} = \frac{1}{2\pi}$$

### 3.3 Verification Through the Zeta Function

We can verify this through the derivative of the Riemann zeta function:

$$\zeta(s) = \prod_p \frac{1}{1 - p^{-s}}$$

The logarithmic derivative at s = 0 gives:

$$\frac{\zeta'(0)}{\zeta(0)} = \sum_p \frac{\ln p}{p^s - 1}\bigg|_{s=0}$$

Using ζ(0) = −1/2 and ζ'(0) = −½ ln(2π):

$$\sum_p \ln p = \ln(4\pi^2) \quad \text{(zeta-regularized)}$$

confirming that ∏_p p = 4π².

### 3.4 Alternative: Additive Regularization

An alternative regularization scheme works with the logarithms directly. Define:

$$\ln\alpha_{\text{adelic}} = \sum_p \ln\alpha_p + \ln\alpha_\infty$$

With ln α_p = −½ ln p and zeta-regularized Σ ln p:

$$\sum_p \ln\alpha_p = -\frac{1}{2}\sum_p \ln p = -\frac{1}{2}\ln(4\pi^2) = -\ln(2\pi)$$

Then:

$$\ln\alpha_{\text{adelic}} = -\ln(2\pi) + \ln\alpha = \ln\left(\frac{\alpha}{2\pi}\right)$$

$$\alpha_{\text{adelic}} = \frac{\alpha}{2\pi}$$

This reproduces the same result through an independent regularization path.
